### Phase 3: Running the Optimization (The Heavy Lifting)

**6. Configure Optuna and ASHA (Step 2.3 & Part 3)**
*   Set up the Tuner. The assignment hints at using the **ASHA Scheduler** to terminate bad runs early and save time.
    ```python
    from ray.tune.search.optuna import OptunaSearch
    from ray.tune.schedulers import ASHAScheduler

    # Search Algorithm
    optuna_search = OptunaSearch(metric="loss", mode="min")
    
    # Early Stopping Scheduler (The Efficiency Challenge)
    asha_scheduler = ASHAScheduler(
        time_attr='training_iteration',
        metric='loss',
        mode='min',
        max_t=20, # Max epochs per trial
        grace_period=5, # Minimum epochs to run before killing a bad trial
        reduction_factor=2
    )

    tuner = tune.Tuner(
        train_tune,
        tune_config=tune.TuneConfig(
            search_alg=optuna_search,
            scheduler=asha_scheduler,
            num_samples=20, # The assignment requires 20 combinations
        ),
        param_space=search_space
    )
    ```

**7. Run the Sweep**
*   Execute `results = tuner.fit()`. 
*   *Warning:* This will take a while as it trains 20 different models (though ASHA will kill the bad ones early).
